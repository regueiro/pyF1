establecer_directorio_bd(path)

Configura el directorio en que se encuentran los ficheros yaml que conforman la BBDD de la aplicación.

Parameters:
  • path (string) - path al directorio.

obtener_lista_temporadas()

source code 

Devuelve una lista con los nombres cortos de todas las temporadas almacenadas en la BBDD.

Returns: [String]
Lista de nombres cortos de temporadas.
Raises:
  • IOError - Si ocurre algún error en el sistema de ficheros al acceder a la BBDD.

Note: La BBDD está formada por los ficheros yaml en el directorio correspondiente

See Also: Facade.establecer_directorio_bd

obtener_temporada(nombre_corto)

source code 

Devuelve el objeto correspondiente a la temporada cuyo nombre corto se especifica. Si el objeto no se encuentra en memoria, se recupera de la BBDD. Al recuperar una temporada, también se recuperan todos los objetos relacionados.

Parameters:
  • nombre_corto (String) - El nombre corto de la temporada. @see{nueva_temporada}.
Returns: Temporada
El objeto temporada correspondiente.
Raises:
  • IOError - Si ocurre algún error en el sistema de ficheros al acceder a la BBDD.

almacenar_temporada(temporada)

source code 

Se almacena en la BBDD el objeto temporada y todos los objetos relacionados con él.

Parameters:
  • temporada (Temporada) - El objeto temporada a alamacenar.
Raises:
  • IOError - Si ocurre algún error en el sistema de ficheros al acceder a la BBDD.

Note: El objeto temporada debe tener un nombre_corto válido @see{nueva_temporada}

eliminar_temporada(temporada)

source code 

Elimina de la BBDD la temporada y todos los objetos (GP, Piloto, Equipo) asociados a esa temporada.

Parameters:
  • temporada (Temporada) - El objeto Temporada que se eliminará.
Raises:
  • IOError - Si ocurre algún error en el sistema de ficheros al acceder a la BBDD.

source code 

Crea una nueva temporada. Devuelve el correspondiente objeto de tipo Temporada.

Parameters:
