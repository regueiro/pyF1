puntos_piloto(self, piloto, gp=None)

source code 

Devuelve los puntos obtenidos por un piloto. Si se especifica un GP, devuelve los puntos en dicha carrera, en caso contrario devuelve el acumulado en toda la temporada.

Parameters:
  • piloto (Piloto) - El piloto del que se consulta la puntuación.
  • gp (GP | None) - El gran premio para el que se consulta la puntuación o None para el acumulado de toda la temporada.
Returns: int
La puntuación consultada.
Raises:
  • ValueError - Si se solicita la puntuación de un GP que todavía no ha finalizado.

establecer_datos(self, datos)

source code 

Actualiza los siguientes datos de la temporada: nombre, nombre corto, fecha de comienzo, fecha de finalización, número de pilotos por GP, número de pilotos clasificados, y tabla de puntuaciones.

La fecha de inicio y las reglas no se pueden actualizar después de que haya comenzado la temporada, i.e. al menos un GP se ha disputado.

Sólo se actualizan aquellos datos para los que se proporciona un valor.

Parameters:
  • datos (dict) - Los datos a actualizar, en un diccionario con las siguientes claves: 'nombre', 'nombre_corto', 'inicio', 'fin', 'n_pilotos_gp', 'n_pilotos_parrilla', 'tabla_puntos'.
Raises:
  • Forbidden - Si el diccionario contiene alguna clave invalida.
  • BussinessRuleError - Si se intenta cambiar la fecha de inicio o las reglas de carrera después de que se comienzen a disputar los GPs.
