Class Piloto

util.Obervable --+
                 |
                Piloto

Los objetos de la clase Piloto se utilizan para representar a los distintos pilotos que participan en la competición.

Algunos de sus datos relevantes son el nombre y la fecha de nacimiento. Opcionalmente puede estar disponible una foto, para la cual se indica un uri que permita recuperarla la imagen.

Durante una temporada, un piloto puede cambiar de equipo, por lo que se mantiene la referencia a todos los equipos a los que estuvo vinculado, junto con la fecha de inicio de la relación.

nuevo_equipo(self, equipo, desde)

 

Vincula al piloto a un nuevo equipo a partir de la fecha indicada.

Parameters:
  • equipo (Equipo) - El nuevo equipo del piloto.
  • desde (datetime.date) - Fecha de inicio de la relación.

eliminar_relacion(self, equipo, fecha)

 

Permite eliminar la vinculación establecida entre el piloto y el equipo en la fecha dada.

Parameters:
  • equipo (Equipo) - El equipo al que está/estuvo vinculado el piloto.
  • fecha (datetime.date) - Fecha de inicio de la relación.
Raises:
  • ValueError - Si no existe la relación con el equipo, o no comenzó en la fecha indicada.

obtener_equipo(self, fecha=None)

 

Recupera el equipo al que se encontraba vinculado el piloto en la fecha indicada, si existe.

Si no se especifica la fecha, devuelve el último equipo, i.e. el equipo actual.

Parameters:
  • fecha (datetime.date | None) - La fecha para la que se desea conocer la vinculación, o None obtener el más reciente.
Returns: Equipo
El equipo al que se encontraba vinculado el piloto. None si ninguno.

estuvo_en_equipo(self, equipo, periodo)

 

Determina si el piloto está o estuvo vinculado al equipo en algún momento del período comprendido entre las fechas dadas.

Parameters:
  • equipo (Equipo) - El equipo para el que se realiza la consulta.
  • periodo ((datetime.date, datetime.date)) - Fechas de comienzo y fin del período.
Returns: bool
True/False si el piloto se encontraba vinculado al equipo en el período dado.

establecer_datos(self, datos)

 

Actualiza los siguientes datos del piloto: nombre, fecha de nacimiento y uri de la foto.
