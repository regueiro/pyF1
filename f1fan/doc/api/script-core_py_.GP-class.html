Class GP

           object --+    
                    |    
model.util.Observable --+
                        |
                       GP

Los objetos de la clase GP representan los Grandes Premios de una temporada. Los datos de un GP son su nombre, lugar de celebración, fecha de comienzo y fecha de finalización.

Para los GPs que ya se han disputado, se mantienen sendas relaciones con el orden en que finalizaron los pilotos la sesión de clasificación y el orden en que finalizaron la carrera.

Así mismo para cada GP finalizado es necesario indicar el % de vueltas completadas, ya que esta información es necesaria para el cálculo de las puntuaciones.

Actualiza los siguientes datos del GP: nombre, lugar, fecha de comienzo, fecha de finalización, número de vueltas a completar.
