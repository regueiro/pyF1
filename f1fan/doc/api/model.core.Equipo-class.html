Class Equipo

source code

     object --+    
              |    
util.Observable --+
                  |
                 Equipo

Los objetos de la clase Equipo se usan para representar los distintos equipos que participan en la competición de F1.

Los datos relevantes para cada equipo son su nombre, fecha de comienzo en la competición, y fecha de fin, en caso de que el equipo se haya disuelto o haya abandonado la competición.
